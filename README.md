# tinyedm

Minimum implementation of EDM (Elucidating the Design Space of Diffusion-Based Generative Models) on cifar10 and mnist.

## Installation

```
git clone https://github.com/YichengDWu/tinyedm.git
cd earth2mip && pip install .
```
